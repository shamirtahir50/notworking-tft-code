FAQ

How to resize the image?

  • Click the Browse... button to select the image.
  • In the Resize Image section, enter its new dimensions in pixels in the Width and Height fields or use the Scale Percentage setting. To change the image proportion, unselect the Lock aspect ratio option.
  • Click the Save Image button to download the resized image.

To resize the image in KB or MB, check the RedKetchup - Image Compressor.

How to crop the image?

  • Click the Browse... button to select the image.
  • In the Crop Image section, adjust the highlighted area on the canvas, or enter values in the Left, Top, Width, and Height fields. It is also possible to automatically adjust the area using one of the Aspect Ratio options.
  • Click the Save Image button to download the cropped image.

How to resize the image to a specific aspect ratio?

In the Crop Image section, select one of the available ratios from the Aspect Ratio menu or add a custom one. Then click the Resize image to aspect ratio option to resize the image instead of cropping it.

How to center the image using transparent margins?

When the image is resized to a new aspect ratio, instead of stretching the image, it is possible to center it using transparent margins.

Configure the option Fill Mode and save the image as PNG, WEBP or HEIC to keep the transparency.

How to resize multiple images at the same time?

It is possible to compress, convert, and resize multiple images in batch with the RedKetchup - Bulk Image Resizer.
